# DATA  
- String (Any charecter or text that wrap in **single** `('')` or **double** `("")` quote)
    - Example
       - `"Pizza"`
       - `"Burger"`
- Number(Any numerical value)
    - Example
       - `10`
       - `10.5`
- Boolean (Similar to yes or no , 1 or 0)
    - Example(no quotes required)
      - `true`
      - `false`
